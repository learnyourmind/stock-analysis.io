# stock-analysis.io

## Stock Analysis 

### Controls you can tune before generating:

Toggle any of the 16 sections on or off
Switch between Standard / Deep Dive / Ultra (initiation of coverage) depth
Choose Structured, Narrative prose, or Executive Brief format

The prompt also closes with a mandatory synthesis section that forces a rated conclusion (Buy/Hold/Sell), 12-month price target with a high/low range, and position sizing guidance — the deliverable that separates a research report from a commentary piece.
